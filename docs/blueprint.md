# **App Name**: ToyVerse

## Core Features:

- Category Showcase: Display various toy categories with visually appealing layouts.
- Product Listing: List multiple products within each category with details and images.
- Add to Cart Functionality: Enable users to add selected toys to a virtual shopping cart.
- Checkout Form: Collect basic user details (name, email, WhatsApp number) via a form at checkout.
- WhatsApp Floating Icon: Implement a floating WhatsApp icon for direct customer support.

## Style Guidelines:

- Primary color: Vibrant blue (#3498DB) to evoke trust and playfulness.
- Background color: Soft, desaturated blue (#EBF5FB) for a calming backdrop.
- Accent color: Energetic orange (#F39C12) to highlight calls to action and important elements.
- Body and headline font: 'PT Sans' (sans-serif) to keep a modern look with a touch of warmth. It's very readable for both headlines and body text.
- Use bright, custom icons that represent each toy category. The WhatsApp icon should be the standard green.
- Clean and responsive design that adapts to different screen sizes. Prominent search bar for easy navigation.
- Subtle animations on product hover and cart updates to provide user feedback and a playful experience.
